# TileMapTool

A single-file, [no-dependency](http://vanilla-js.com/), locally-runnable tool for exploring tile maps.

## Usage

### Locally

Just clone this repository and open `index.html`. That's it.

### Online

[Here](https://tmt.computerpho.be/)
